# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-01

### Added
- Complete documentation structure for all homelab infrastructure services
- Documentation for pfSense, Switches, Proxmox, and Terraform
- Documentation for Vault, Media services (Plex, ServArr stack)
- Documentation for Web Services (Nginx, Ghost, MySQL)
- Documentation for Kubernetes deployments
- Documentation for DNS, SMB, WiFi, and LLM services
- Complete directory structure documentation
- Architecture overview and network design documentation

### Changed
- Updated README.md with comprehensive overview
- Enhanced CONTRIBUTING.md with detailed guidelines
- Added CODE_OF_CONDUCT.md and SECURITY.md files
- Standardized documentation format across all services

### Fixed
- Improved cross-referencing between documentation sections
- Corrected formatting and consistency issues
- Enhanced security practices in documentation

## [0.0.1] - 2026-02-17

### Added
- Initial repository structure
- Basic documentation for pfSense and Proxmox
- Initial documentation for Vault and Traefik
- README.md template
- AGENTS.md with documentation guidelines
